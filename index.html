Bienvenue sur l'alpha du projet "La tête dans le Cube" !

Notre projet est simple : centraliser toutes les informations sur le cube en un seul site, afin que chacun y trouve ce qu'il cherche, qu'il soit débutant, professionnel, collectionneur de cubes, moddeur ou même fan de maths.
Un projet ambitieux n'est-ce pas ?
Le site n'en est encore qu'à ses débuts mais il progresse, lentement mais sûrement...

Si vous avez des questions, vous pouvez nous les poser dans le topic associé sur le forum francocube, ou bien nous contacter avec la rubrique "Nous contacter" dans le menu à gauche.


Sur la page Journal de bord, vous pourrez voir un petit compte-rendu régulier ainsi que certains fichiers intéressants.
